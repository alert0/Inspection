关于进一步规范信访举报件办理工作的

通  知


各镇纪委，各街道纪工委，区各委办局（公司）、区各直属单位纪委（纪检组），委局机关各室：

为进一步加强和规范信访举报件办理工作，完善归口受理、规范办理、闭环管理工作模式，提升信访举报件办理质量，现将有关事项通知如下：

一、统一归口受理信访举报

区纪委信访室归口受理同级党委管理的党组织和党员干部违反党纪的信访举报，统一接收下级纪委和派驻纪检组报送的相关信访举报，按照程序由区纪委信访室受理、登记，经分管领导或主要领导审签后分流办理，分类摘要移送案件监督管理室、干部管理监督室，并定期跟踪移送信访举报件处置情况。

二、建立督办工作责任体系

建立区纪委主要领导负总责、分管领导重点抓、信访室具体负责、各纪检监察室和镇、街道纪委（纪工委）、区各直属单位纪委（纪检组）共同参与的信访举报件督办工作责任体系，明确督办责任人员，形成有力组织保障。委局机关各承办科室、各镇、街道纪委（纪工委）、区各直属单位纪委（纪检组）要严格落实信访举报转送办理情况反馈要求，将《问题线索处置登记表》经审批后2日内报区纪委信访室备案，按照规定时限及时将信访件办理结果反馈信访室。（一般信访案件采取谈话函询和初步核实方式处置，结案时限为2个月;采取立案审查方式处置，结案时限为3个月。）

按照“谁办理，谁反馈”的原则，具体承办人员在信访举报件办理作出结论7个工作日内，将处理结果以适当方式告知举报人，听取并记录举报人的意见，填写《贾汪区纪委监察局署实名举报反馈单》，对于举报人不满意处理结论的，认真分析研究，视情补充调查，并将调查结果和处理意见再次向举报人反馈。《实名举报反馈单》报信访室留存。区纪委信访室对实名举报、越级访、重复访的信访件进行重点督办，不定时抽查办理情况，对办理不认真、反馈不及时的将适时通报，情节严重的根据规定追究相应责任。

三、实行查否审核制度

实行查否审核制度，对区纪委主要领导签批的、上级要结果的、巡视巡察交办的、向反映人反馈后不满意并反复上访的、越级访的等五类信访件实行查否审核。在信访件办理结束后，各承办科室要填写《信访件查否审核意见表》，审核材料要齐全、完备，由本科室负责人提出处理意见，报分管常委、分管副书记审核后，经审理室、信访室复审，根据干部管理权限，报分管副书记、主要领导审核，填写《信访案件结案呈批表》后结案。经审查问题未查清的继续调查，对于查否的信访件，后因举报人重复举报另行调查又查出问题的,追究原主办人责任。

以上事项，依据《2017年度贾汪区纪委、监察局机关党风廉政建设和反腐败工作监督责任考核办法》、《2017年度机关单位纪检监察组织党风廉政建设和反腐败工作监督责任考核办法》、《2017年度镇、街道纪检监察组织党风廉政建设和反腐败工作监督责任考核办法》进行考核。
