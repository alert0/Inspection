贾汪区：“纪检干部课堂”为干部充电补能增力

2017-01-01

阅读量：33

 “要把握好操办婚丧事宜的几个关键：一是是否利用职权或者职务的影响，二是是否占用了公车、公款等，三是是否造成不良影响，四是是否宴请了管理服务对象，5是是否大操大办，6是是否借机敛财....”这是近日贾汪区纪委案件审理室副主任在该区“纪检干部课堂”给全区纪检监察干部上课的内容之一。


为扎实践行“打铁必须自身硬”要求，贾汪区纪委开设“我的业务我来讲”纪检干部课堂，以常态化学习促使纪检干部补齐短板，不断提升监督执纪问责的能力和水平。据了解，“纪检干部课堂”每周举办一次，每次安排一至两个委局机关科室围绕本科室主要业务，自定课题、自备课件、自当讲师，在引导纪检干部学习、交流、进步的同时，也促进使了“讲师们”在授课中反思、回顾、总结。


“紧贴实际”、“针对性强”“量小而精”是纪检干部课堂的鲜明特点，每堂课原则上不超过2个小时，授课内容的设置以该区党风廉政建设和反腐败工作开展的实际需要和该区纪检干部的实际短板为主要依据，课题、讲课稿和课件须提前向干部管理监督室报备，并经室主任、分管领导层层把关后，才能最后正式进入“课堂”。同时每堂课的授课人还需按要求公开联系电话、微信号以及QQ号，以方便学员围绕不懂的问题与授课人进行课后讨论。“课上+课下”的授课方式改变了过去传统式、灌鸭式的教学模式，增加了交流性、互动性元素。

“派驻机构改革后我首次进入了纪检系统，面对全新的工作，我因知识的匮乏而感到强烈的本领恐慌，而此次培训就是一场及时雨，为我提高务水平和执纪能力，切实履行好监督执纪问责主业提供了支撑。”该区派驻第四纪检组工作人员乔方无不激动的说到。

截止日前，该区已有三名纪检业务骨干登台授课，取得了很好的教育效果，下一步该区将结合授课中的具体实践，进一步谋求授课形式和授课内容的优化，将其打造成贯彻“打铁必须自身硬”要求，全面提升纪检监察干部队伍整体素质的又一项品牌工程。
