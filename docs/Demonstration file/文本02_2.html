《徐州市纪检监察干部外出报备制度》

2017-10-27

阅读量：33

徐州市纪委日前出台《徐州市纪检监察干部外出报备制度》，明确外出报告对象，规范外出审批程序，进一步严肃外出报备等组织纪律，加强和规范对纪检监察干部的监督管理。

　　哪些纪检监察干部外出需要报备?该制度首先明确了外出报备对象：市纪委监察局机关、市纪委派驻纪检组、市委巡察组全体干部，各派出纪工委书记、各县(市)区纪委书记，因私离开本市1天以上的应提前履行书面报告审批手续，未经批准同意的，不得擅自外出。

　　此项制度还明确规范了外出审批程序。该制度规定，市纪委监察局班子成员、各县(市)区纪委书记、各室主要负责人、巡察办主任及副主任，因私离开本市的，向市纪委主要领导请示报告;各派出纪工委书记、各派驻纪检组组长、市委巡察组组长及副组长，因私外出向委局分管或联系领导请示报告;其他纪检监察干部因私外出向部门主要负责人请示报告。请示报告经同意后，填写《徐州市纪检监察干部外出报备情况表》送市纪委干部监督室备案。

　　通过此项制度的制定实施，市纪委得以进一步严肃外出报备纪律。据了解，市纪委干部监督室将对制度执行情况进行监督检查，并视情况通报。对外出或外出逾期不请示报告的干部，进行通报批评;对造成不良影响或严重后果的，追究党纪政纪责任。
