恪尽职守　廉政为民

――塔山镇纪委先进事迹材料


近年来，塔山镇纪委聚焦监督、执纪、问责“三大主业”，实践监督执纪“四种形态”，持之以恒纠“四风”，把全面从严治党不断向基层、向一线、向群众身边延伸，着力打造政治生态上的“青山绿水”，2016年以来，立案26件，结案26件，给予党纪处分26人，其中开除党籍1人，严重警告处分3人。

（一）强化政治担当，夯实治党责任。认真贯彻落实党委主体责任,镇党委书记每年初都要与10个村党组织签订党风廉政责任书，每年镇党委研究党风廉政建设工作不少于3次，党委书记亲自安排部署党风廉政建设不少于4次。明确党政领导班子成员“一岗双责”，镇党委书记与12名镇党政领导班子成员签订从严治党“一岗双责”责任状。认真开展“党委书记月月谈”，镇党委书记先后对彭楼村支部书记权超、聂庄村支部书记付海文及粮管所所长李道坤等15名村、单位负责人进行了谈话，达到了“红红脸、出出汗”的目的。

（二）丰富宣传教育，营造廉洁氛围。组织监察室和纪委委员认真学习党的十八大、十九大精神和习近平总书记系列重要讲话精神。镇党委班子成员带头加强党纪党规学习，组织镇村两级干部100余人观看了警示教育片，转发了典型案例通报2期。投资32万余元在10个村建成10个大数据信息平台，随时为群众提供各项资金、信息等查询。

（三）践行“四种形态”，遏制腐败蔓延。聚焦主责主业，全面查处基层处事不公、以权谋私、作风不实的问题，重点突出对扶贫领域专项资金的监督执纪问责，组织农经中心、审计所等对全镇建档立卡户、扶贫资金、惠农补贴等进行专项核查，全面清理农村集体资产、资金、资源等，严查农村基层不正之风。　

（四）持续纠正“四风”，祛除作风顽症。持之以恒纠正“四风”问题，从不迟到、不早退、不走读、开会不玩手机等作风建设小事抓起，持续发力，抓紧抓实，确保作风建设规范化、常态化、长效化。截止目前，对机关效能和作风建设开展常规检查和经常性抽查20余次。　

（五）扎实开展“三清示范区”建设 。 把“三清示范区”建设与“两学一做”活动有机结合，严格按照《2016－2021年贾汪区项目化推进廉洁政治“三清示范区”建设方案》的标准和要求，抓好这塔山、殷庄两个示范村的建设，同时，以点带面，在各村进行逐步推广。

（六）认真执行“1＋2”文件，打好严管村干部组合拳。组织全体班子成员和村干部认真学习《贾汪区严管干部实施意见》、《贾汪区村干部及其近亲属享受非普惠性惠民政策提级审核实施办法》和《贾汪区村干部负面清单》三个文件。特别对村干部十条“负面清单”、九条“非普惠性惠民政策提级审内容”及违纪违法行为需要给予党政纪处分的八个情形进行了强调。　

行到深处是坚持。塔山镇纪委将以永远在路上的坚定决心和顽强品质，秉承“零容忍”的态度、“钉钉子”的精神，持久抓、深入抓，巩固、深化来之不易的党风廉政建设成果，厚植经济发展的政治基础。

2017年11月7日
