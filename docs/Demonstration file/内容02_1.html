2016年工作总结

党风政风监督室

 今年以来，在区纪委的领导下，党风政风监督室聚焦主责主业，强化担当作为，深入推进全区党风政风监督工作，取得了一些成绩。现将情况总结如下：


（一）明确责任抓落实，深入推进党风廉政建设

压实“两个责任”，推进党风廉政建设责任落实。与基层党委签订党风廉政建设责任书，明确职责，压实责任。制定《贾汪区落实<中国共产党问责条例>实施办法》，办法出台以来，共通报批评14家党委，诫勉谈话24人。以挺纪在前“五大行动”为抓手，树牢纪律观念。按照全面从严治党要求，坚决把纪律和规矩挺在前面，开展挺纪在前“五大行动”，30名科级干部申报和报告了婚丧喜庆事宜。

（二）开展各类专项行动，持续推进作风建设

开展违规吃请和公款吃喝问题专项整治工作，查处违规吃请问题1件，党纪处分1人，诫勉谈话4人。开展扶贫领域监督执纪问责工作专项督查，立案4件，党政纪处理8人。持续推进节日期间作风建设工作。节日期间，对公车使用、公款吃喝等问题进行明察暗访14次。严查群众身边的“四风”和腐败问题。查处6起，党政纪处分6人，移送司法机关1人。加大对服务窗口单位工作纪律情况开展专项检查。出台《关于进一步严肃执行领导干部请假值班制度和纪律的通知》，念好全区党员领导干部作风建设的“紧箍咒”。牵头开展全区巡察工作。完成对青山泉、江庄、紫庄等8家乡镇、街道巡察工作。

（三）突出重点抓创新，拓展工作新领域

牵头开展优化营商环境相关工作。健全完善相关制度规范，出台《中共贾汪区委贾汪区人民政府关于进一步优化营商环境的实施意见（试行）》、《贾汪区营商环境问题举报奖励办法（试行）》。加大对破坏营商环境行为的曝光力度，下发通报2期，党政纪处分6人，诫勉谈话9人。制作警示教育短片，丰富监督执纪方式。制作《作风建设永远在路上——贾汪区作风建设明查暗访实录》视频短片。用身边事警示教育身边人，提醒广大党员干部不越“雷池”，不踩“红线”。
