严格监督管理。做好委局机关公开选调公务员相关工作，及时为新录用的4名公务员办理调动手续。落实我区纪检干部参与省市培训的对接工作，12月份组织部分纪检监察干部赴杭培训。参与制定印发《贾汪区领导干部任前廉政法规考试实施办法》，组织开展任前廉政法规考试2次，参考人员19名。严格把好党员领导干部“党风廉政意见回复”关，2016年函复52次。加强涉纪涉腐网络舆情分析研判，全年编发舆情专报12期。对1起反应我区纪检监察干部线索问题进行函询。全力做好区纪委常委会和党支部换届服务工作，按要求申报各类表格、文字材料20余份。完成省级纪检监察系统先进个人、公务员年报等30余项材料的申报工作。
