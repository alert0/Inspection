2016年以来，本人在区纪委、街道党工委的正确领导和精心指导下，认真贯彻落实习近平总书记系列重要讲话精神，结合“两学一做”学习教育活动，严格落实党风廉政建设责任制，进一步改进工作作风，提高机关效能,增强服务意识,为促进潘安湖街道事业发展提供了有力的政治和纪律保障。

（一）全面落实党风廉政建设责任制、落实考核机制。协助将党风廉政建设和反腐败工作纳入经济社会建设中，落实党工委书记对职责范围内的党风廉政建设负全面领导责任，班子其他成员认真履行“一岗双责”，确保潘安湖街道党工委的主体责任、纪工委的监督责任落实到实处。制定了《廉政风险防控实施方案》，对班子成员党风廉政建设责任目标进行分解。（二）不断加强党的作风建设，落实监督责任。认真贯彻落实中央“八项规定”，省委、市委“十项规定”和区委“八项规定”，持而不息纠正“四风”。制定了《潘安湖街道机关作风督查和投诉受理工作制度》，全面开展各类纪检专项行动，深化“百姓办事零障碍”工程，九个村居全部建成便民服务室，打通联系服务群众最后一公里。整治损害群众利益的不正之风，深化村级“勤廉双述”、“三资四化”等制度，加大村务公开、财务公开力度。对扶贫领域、农村集体“三资”管理进行重点监督。对营商环境进行了重点监督，出台《潘安湖重点工程推进实施办法》，在环境保护、秸秆禁烧、263专项行动、环境整治，清洁家园专项行动、重大项目建设中加强对权力行使的监督制约，扎紧制度的笼子。（三）强化对党员领导干部的监督，做好“森林防火员”。在潘安湖街道实行领导干部报告个人有关事项制度；严格落实党员干部婚丧喜庆事宜报告制度；严格做好“三公”经费管理，从严控制行政经费支出。全年开展廉政谈话20余次。在人大代表、村委换届选举中严明纪律，对群众举报的线索快查快办，为换届有关工作健康有序开展提供了政治保障。（四）贯彻党的精神，丰富宣传形式，加强对党的纪律、方针政策和决议决定执行情况的监督。结合“两学一做”主题教育活动，开展观看了《尺度》、《永远在路上》、“守住底线、不忘初心”大讨论等廉政教育活动；各村居建设廉政宣传栏，丰富了廉政宣传内容。2016年建成清心居一般谈话室、廉政文化广场3处（马庄村、权台村、荷香园）、马庄村“三清示范点”展示馆，农民党员自发编制快板《勤政廉政见实效》等都极大地营造了健康的党内政治生活局面，潘安湖街道打造出一支为民、务实、清廉、高效的干部队伍。（五）运用“四种形态”，严肃查处违纪违法案件。严肃处理了一批违法违纪和损害群众利益的人员。2016年总计立案17件，给予党纪轻处分16人次、开除党籍1人次，信访排查16次，开展廉政谈话、约谈20余次，通报批评3人次，让“咬耳扯袖、红脸出汗”成为常态。（六）认真推进“三清示范区”建设。以“目标工程化、工程项目化、项目责任化”为工作思路，牢固树立从严治党永远在路上的思想，全面开展廉洁政治“三清示范区”建设工作。
