下一步，区纪委将按照活动要求，按照序时进度，把握节奏、跟踪帮扶，统筹做好单位之间的协调、联络，严格按照纪律要求引导好、管理好党员干部，理清工作思路，化难题、保发展、促和谐，确保帮扶活动见真章、出实效。“百家机关联百镇、千名干部进千村、万名党员入万户”集中走访帮扶活动开展以来，贾汪区纪委始终坚持以密切党群干群关系，服务推动市委、区委中心工作为宗旨，牢牢把握群众工作路线，严格按照步骤要求扎实开展帮扶任务。现将区纪委1-3月份工作任务完成情况汇报如下：

一、精心组织部署，强化责任担当

活动开展以来，按照区委、区政府的工作部署，及时传达上级指示精神，强化组织领导，严格对照标准，精心做好走访信息统计任务。

一是迅速传达落实。第一时间召开委局机关全体党员会，迅速传达上级精神，深入学习活动方案，教育引导党员干部深刻认识理解此次的活动的重要性，把走访帮扶的过程作为接受教育、转变作风的过程。认真做好联络沟通、上传下达的工作任务，组织召开区级包挂单位联席会议，动员相关单位积极参与帮扶活动，确保各单位按规定动作参与活动。

二是精心组织筹备。按照区委、区政府的任务部署，在各单位内部深入贯彻落实上级精神，发动党员27人，成立9个工作小组，扎实推动各项工作有序开展。各单位严格对照上级要求，结合各单位实际，成立工作小组，制定工作方案，动员广大党员干部高标准出色完成任务。

三是扎实完成统计工作。活动开展以来，区纪委作为集中走访帮扶活动的区级包挂牵头协调单位，动员其他9家联系单位，严格按照上级指示精神，梳理问题清单，截至目前，累计梳理清单211条。

二、密切联系实际，深入贯彻落实

一是根据实际情况，分组走访调研。根据9个小组实际，动员本单位党员深入调研村组实际情况，分组包片调研走访。二是做足前期功课，精准实施走访。严格按照方案及既定表格要求，提前了解掌握所在辖区人口数量与居民可能存在的困难情况，在走访前深入开展调研，做到“群众所需我了解、群众所求我掌握、群众困难我能解”。三是突出问题导向，着力化解难题。按照既定方案，围绕“补短板帮民富、解难题助发展、化矛盾促和谐”工作目标，重点解决群众思想困惑、群众生活困难、项目建设障碍、企业发展瓶颈、影响社会稳定、基层党建薄弱六大难题。

三、结合自身特色，传递基层声音
