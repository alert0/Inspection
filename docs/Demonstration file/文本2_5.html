从君子九思谈党员干部的忠诚干净担当


袁邦庆

（2017年6月30日）

根据录音整理

  结合“两学一做”活动和“打铁还需自身硬”的专项行动来给大家上一堂党课。我今天上的党课题目是从“君子九思”来看如何做忠诚干净担当的党员领导干部、纪检监察干部。

  习近平同志在2014年在河南兰考考察时告诫领导干部：领导干部要自觉从中华民族的优秀文化中汲取营养，老老实实的向身边群众学，时时处处见贤思齐 ，以严格的标准加强自律，接受他律。从他这段讲话中我们可以看出领导干部加强自身学习的四种资源：一、历史传统，二、人民群众，三、典型榜样，四、制度标准。孔子在《论语·季氏》中提出了君子为人处世要做到“九思”，哪“九思”呢,即视思明、听思聪、色丝温、貌思恭、言思忠事思敬，疑思问，忿思难，见得思义。这“九思”对当前我们领导干部，特别是纪检监察干部加强自身修养有很多值得我们借鉴汲取的地方，下面我就对照这“九思”来看看我们党员领导干部如何来加强个人修养：

  第一，视思明。观察事物要彻底明白，做到真明白。我们常说“眼见为实，耳听为虚”，但是有时候我们看到的也不一定是真实的。所以我们看实物要从上下左右，是非对错，多角度去观察，这样才能看到真相。作为纪检监察干部如何才能做到视思明呢？

  我认为首先要做到这三点：第一点站得高，“不畏浮云遮望眼，自缘身在最高层”，在高处才能看到，上山为仙下山为人，人只有登到山顶才能为仙，为仙之后才能神机妙算，那些虚假的东西才能看清，只有登的高才能看得远，也就是要求我们纪检监察干部要有大局观念，中心观念，从党的路线方针政策，从党风廉政建设和反腐败大局去思考问题。大局意识，就不能只站在本单位、本部门角度去思考问题。第二智商要高。孔子说：智者不惑，仁者无忧，勇者无惧。智就是智慧策略，一件事让你去处理，你可能处理的鸡飞狗跳，换一个人去处理，可能就很平静地把事处理好了。我们在讲中庸之道，最精髓的就三个字：度、权、和。什么是度呢，就是做任何事要把握分寸，过犹不及。权，就是权变，善于变通。什么是和，就是和而不同，不能随便糊弄。只有这样，做一名智者，才能透过现象看本质，看清事物的本来面目。不能被表象迷惑，学会抽丝剥茧。只有智者才能视人生如节目，是比较潇洒顺心的，有的人干工作，十分难为，整夜整夜睡不着，什么原因？缺少智慧。第三政治成熟度要高。习总书记对领导要求，要做政治上的明白人。党员领导干部，区别于普通党员，要起到示范作用，这是党员领导干部的一种特质，这种特质在落实政治方向、政治立场、政治鉴别力、政治观念上体现政治纪律和政治敏锐性，做政治上的明白人，我们纪检监察干部要坚定共产主义理想信念，表现在执行党的纪律和规矩上坚定原则，毫不含糊。有个领导干部给我讲过这样一个观点，我认为很有道理，就是一个人政治上成熟不成熟，看有几个活动他参不参加：第一个能不能敢到群众中去，敢不敢接见群众的来访，敢不敢见群众。有的干部怕见群众，没有上访的走得还慢一些，一看上访的来了跑得比谁都快。这就体现出作为一名党员干部，你能不能做到密切联系群众。为什么从中央到省市再到县区，我们都设领导干部定期接访制度，有时候这就是一种姿态。第二个当群众有困难的时候，你是不是心系困难群众。当扶贫济困的时候，一般情况下，领导干部都要参加，这就说明作为党员干部，你是不是把人民的疾苦放在心上。第三个有关党风廉政建设、廉政教育的活动，凡是政治成熟度高的干部，他是一定会去参加的。他心里可能不想去参加，但他只要是政治成熟度高的他就一定会去。凡是不愿意参加党风廉政、廉政教育的干部，他的政治成熟度不高，政治上还没成熟。

  第二，听思聪。听是指听力、听辨。不论听到什么，哪怕是一件事、一种理论、一种观点都不能人云亦云，特别是我们纪检监察干部要学会用自己的智慧去判断，取舍。谣言经过三个人都成真的了。

  我们纪检监察干部如何做到听思聪呢？第一胸怀要广。腹中天地宽，常有渡船人。心胸宽广，不要因为令人不快的事而心烦意乱，失望悲观。特别是自己被人误解、歪曲的时候，要能放得下。有时宽容引起的道德震动要比宽容来的强烈。第二个要善于听。善于听是一种亲和的态度。听长辈的教导，朋友诉说，他人的牢骚，爱人的唠叨，下属的想法。要等人把话讲完，你再评判是非，不要轻易打断别人，这是不礼貌的行为。唐太宗当年问魏征：“人主何为而明，何为而暗？”魏征说：“兼听则明，偏听则暗。”我们在听的时候还要能听出来他说的话的来源在哪里，话外之音在哪里，弦外之音在哪里。我们中国人讲话很艺术的，他的真实想法在表达时未必直白的表达出来。要把弦外之音听出来。第三，闻过即改。别人给你提意见，有错误要改。领导干部有时也是要面子的，但是闻过即改也是领导干部的智慧，决不能将错就错，一错到底。

  第三，色思温。面色要温和，要平易近人。子夏问孔子什么是孝，孔子回答：色难。在父母面前始终和颜悦色很难，你给父母吃的很好，穿得很好，住得很好，这就是孝顺了吗？不是，如果你一天到晚他们脸色看，不能做到和颜悦色，就不算孝。与物质条件比起来，精神更重要。一个人的脸色代表了一个人的内心世界，内心情感。一个人的喜怒哀乐，表现出来的，面对他人，你是和蔼可亲，还是拒人于千里之外还是冷若冰霜，还是烦躁不安，这都靠你的面部表情表达出来。那么如何做到色思温，做到和颜悦色，平易近人？

我认为咱们纪检监察干部应该做到这三点，第一点平民心态，也就是自然的心态。阳光，阴暗，积极，消极都是人不同内心世界和生活态度的反映。老子在道德经上说过这样一句话——归根曰静，静曰不命。意思就是人丛生到死亡肯定是要落叶归根，声音说归根曰静，静曰不命，为什么叫静曰不命呢？就是说归根的本源又产生了新的静，产生新的生命，新的生命有萌动。按照老子的观点，你看我们平时实际生命的最大敌人是什么，我们大家异口同声的肯定会说生命的最大敌人是死亡。但是按照老子的观点，生命的伴侣是死亡，为什么呢，当你出生那一天起，死亡也就伴随着你。死亡是和你生的时候一起过来的。在你的一生当中，生命都是不可确定的，只有一向是可以确定的那就是死亡，所以按照老子的观点吧死亡当做敌人，害怕死亡，惧怕死亡，内心恐惧，是不符合人性的观点的。所以按照老子的观点，生命的伴侣是死亡，相生相伴的关系，那么怎么叫平抑心态，就是说我们特别在面对群众的时候，不要把自己看成，农村的土话叫不得了了，时髦一点就是把自己看做是精英，看做是救世主，说话不打官腔，做事不摆官谱，为人不拿官架子，要以淡然之心看待云卷云舒，花开花落，有的干部就像深闺怨妇，心里有事情，他看不开。自己活得也很累，一天到晚愁的睡不着觉，一看这个提拔了，以看那个还不如他的提拔了，心里面放不下，他就没有平抑心态，也就没有平常心态，这是第一点。第二点要有温和的表情，中庸之道里面怎么说，喜怒哀乐之未发谓之中,发而皆中节谓之和，就是说在喜怒哀乐还没有在脸上表现的时候，这个叫忠，到表现的恰到好处的时候，这叫和。这叫中庸之道的核心，主导思想是天人合一，他倡导的是是天人合一，天道与人道合一，天性与人性和一，理性和情感合一，鬼神与圣人合一。包括五大道三大队，五大道大家都知道，君臣、父子、夫妻、朋友、兄弟。三大队人质，孔子在中庸之道里面也讲到，要达到天人合一还要做九件事情，这九件事是哪九件事呢。修养自身，尊重贤人、爱护亲族、敬重大兄、体恤百姓、劝勉各种工匠、安抚诸侯，优待远方来客。南开中学的创始人叫张伯苓，他的治校就是40个字，哪40个字呢，他实际上就是按照$$排列的，面必净，发必理，脸部要干干净净的，头发要经常理，衣必整，衣服要整理，纽必洁；头容正，肩容平，胸容宽，背容直，你的气质应该是什么样子呢，就是气质气魄叫做无傲无气，你脸色是什么呢，颐和易静，所以你看，我们共和国从这个学校走出来的来的两位总理，你看看，一个是周恩来，一个是温家宝。你看看都是温和，平易近人，谦让的气质，你在那个时候你见过文教宝和周恩来发过大火的，你看他包括面部表情，拿开中学的这个校训很谦让，人家不说吗？谦让就是保持完整，为什么呢？我们大家可能都有体会，你到商店里面去买东西的时候，你看到样品放在那里，你决定买这样商品的时候你肯定会说这样一句话，服务员，你给我拿个新的，我不要这个样品。为什么呢。实际这样样品跟那也是一样的，因为这个样品在橱窗里面天天在展示。他通过展示这个表面变得灰暗了，失去光泽了，你从表面看，好像就了，黑不拉几的，什么原因，天天展示的结果。就是做人要低调，如果为人你也天天展示，你也得退色。天天你展示，显摆。这是第二个。第三要善良的心，特别是我们纪检监察干部，更要有善良的心吗，我经常和大家说，我们执纪监督办案，搞每一项检查，查办每一个案件，一定要以菩萨的心肠去对待他们，哪怕他是犯错误的人，你首先要从内心上本着挽救他的态度，本着为他好的态度去对待他，如果你本身就是执法执纪机关，又是执法执纪人员，你在过一点，或者说再狠一点，那就不好。就是说要善良。第二要有霹雳手段，叫没有霹雳手段，不显菩萨的心肠，对该查办的案件，一定要快准狠，要有善良的新，特别是我们作为纪检监察干部，对干部要是保护的，要有爱护的心。上天我跟张区长开玩笑说这个纪委书记一定要善良，组织部长要狠一点也没事，因为他一天到晚都做好事，找人谈话，都开心，他心狠一点没事，纪委书记如果心狠，纪委书记要善良，这个很辩证的，不要以为纪委就是要去查人，去处理人。我鞥经对有些领导干部说，我们每个领导干部头上都有一个官帽子，不像以前领导干部头上都有一个官帽子，说纪委，纪委第一是看帽的，看帽戴的正不正，偏没偏，歪没歪，执行党的方针政策有没有偏，及时提醒一下，这个叫做看帽子。第二个是掸帽子，看到帽子上面有灰尘了，就要给你掸掸，以前我们包括，四个形态，提醒谈话就是掸帽子，第三，扶帽子，不正了给你扶好，第四个给你拾帽子，就是说你因为某种过失，比如你证走路一阵风把你的帽子挂掉了，实际上过错不在你，拾起来再给你带上，这个就是一些问责，比如上新沂怡地条钢，包括周市长都被处理，周市长他呢就是一阵风来了，吧帽子吹得稍微歪了一点，但是像这样，组织上肯定会给他把帽子给拾起来，再给你带上。最后一点才是摘帽子，你是在不想带哦这个帽子了，你自己就把她摘下来往路边、沟里河里面扔，你这样干脆把帽子摘下来，所以说我们纪检监察干部要有这种心态去对待我们的干部，我一直说我们县区级的行政区域内，达到科级干部确实是不容易，自己付出了多少努力，经理多少竞争，才达到。有的违纪，也有着各个方面的原因。所以纪委一定要本着爱护关心帮助干部的心态去对待我们的党员干部，当然从严执纪有腐必反，有案必查，这个是我们的职责。

第四，貌思恭。貌者就是一个人的仪容仪态，就是说你无论言谈态度，在任何的场合下，在任何的地方，都要给对方一个谦虚、恭敬、尊敬的，恭敬包含两个意思，第一层意思就尊重，你要是看到对人不尊重，从内心里面就不尊敬他，你那还来的对他恭敬？第二个是畏惧，心理有所怕，才恭敬，也叫敬畏，因畏而敬。明朝时候方孝孺说过这样一句话，叫善怕者必心有所正，言有所归，行有所指，偶尔越矩，易不出格，凡是这个人又怕的，凡是又惧怕的人，那一半的身体比较正的吗，讲话都是有规矩的，行动都是适可而止的，偶尔有偏差也不会出什么大事，孔子也说君子有三畏：畏天命,畏大人,畏圣人之言。
