江苏贾汪：以现场教学 重温入党初心

2017-02-09

阅读量：44

为响应“两学一做”号召，传优良传统，做合格党员，3月24日，贾汪区纪委党支部组织委局机关全体党员到淮海战役纪念馆和运河支队纪念馆开展“两学一做”学习教育现场教学活动。

在淮海战役纪念馆，全体党员重温入党誓词，表达了对党的热爱

和忠诚。随后详细了解了运河支队的形成和发展历程，聆听了老一辈革命家和先烈们积极投身革命的光辉事迹。通过实物及图片影像资料，重温革命历史，感受红色经典，领悟革命传统。大家在讲解员的引导下，认真听取了朱村革命历史的讲解，纷纷对革命先烈英勇战斗、浴血奋战、顽强拼搏的英雄精神，以及沂蒙人民踊跃参军、拥军支前、慷慨赴敌的英雄壮举表示敬佩。

参观结束后，党员干部们纷纷表示，在以后的工作中要学习老一辈无产阶级革命家爱民为民情怀，学习党的群众工作优良传统，坚定信念，纯洁党性，以昂扬的精神状态、饱满的工作热情、务实的工作作风来更好的履职尽责，用实际行动开展“两学一做”学习教育，争做合格党员。
