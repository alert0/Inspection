对“不忘初心，牢记使命”的思考

2017-11-16

阅读量：31

 十九大大会的主题是：不忘初心，牢记使命，高举中国特色社会主义伟大旗帜，决胜全面建成小康社会，夺取新时代中国特色社会主义伟大胜利，为实现中华民族伟大复兴的中国梦不懈奋斗。

 不忘初心，方得始终。习总书记在十九大报告中明确指出：中国共产党人的初心和使命，就是为中国人民谋幸福，为中华民族谋复兴。这个初心和使命是激励中国共产党人不断前进的根本动力。

历史和现实都告诉我们，密切联系群众，是党的性质和宗旨的体现，是中国共产党区别于其他政党的显著标志，也是党发展壮大的重要原因，能否保持党同人民群众的血肉联系，决定着党的事业的成败。

今年我23岁，党龄3年。大学时期，入党是一件无比骄傲与自豪的事，为了不愧于党员这枚勋章，我在大学生活的每个方面都更加努力，保持党员的先进性、纯洁性，处处争当榜样。

 大学毕业，我考入江庄镇人民政府工作。我是个从小在城市长大的孩子，在乡镇的这一年，我从事了很多岗位，也真正的了解不同人生的酸甜苦辣,这更加坚定了我的初心：为群众能尽我所能。

我为贫困大学生领到雪中送炭的助学金而欣喜,为他们亲切的叫我“姐姐”充满成就感；我为村里的小队长，一位跛脚老人带着我落实扶贫走访挨家挨户不辞辛苦而感动，更将其树立为榜样，时刻提醒着自己的初心。我喜欢走在田间地头脚踏实地的感觉，我喜欢尽我所能为百姓排忧解难的充实，正是这种喜欢，让我成长，让我忘记了劳累，让我坚定了初心。

我们青年人，生活在这个时代是幸福的。我很幸运的成为深化纪检监察体制改革的切实参与者，成为一名纪检监察干部。

 纪检监察机关承担着维护党纪的重要职责，尤其要始终牢记初心和使命。习近平总书记指出不得罪成百上千的腐败分子，就要得罪十三亿人民。现在的我就要“独立不惧”，就要“心怀九心”，就要当一只“啄木鸟”，把“病树”治好，把“烂树”砍掉。

“打铁更要自身硬”。我要不断提高政治站位。忠诚于党、忠诚于纪检事业，旗帜鲜明讲政治，不忘初心跟党走，自觉履职尽责、努力做廉洁奉公、严于律己的表率，以坚强的政治定力激发工作动力和活力。

我要提升能力素质。一方面，抓好纪检监察业务学习，尽快熟悉应知应会业务知识；认真学习区纪委派驻机构《工作规定》和考核办法，对照新工作新任务新要求，准确认识和把握角色定位。另一方面，在组长的带领下，通过书面材料、逐个科室座谈等形式，向驻在部门的领导和同志们虚心请教，了解各科室主要职能及具体业务，深入学习，补齐知识“短板”，梳理廉政风险点。

我要锤炼优良作风。进一步提升工作执行力，不讲价钱，不打折扣，高质高效的完成组织交办的任务，把纪律、规矩挺在前面，模范遵守党纪,多方面提高自身素质。

青年兴则国家兴，青年强则国家强。中华民族伟大复兴的中国梦终将在一代代青年的接力奋斗中变为现实。

我将努力践行忠诚干净担当的庄严承诺，切实履行监督执纪问责的神圣职责，永远保持一颗赤子之心，为我们美丽贾汪、美丽祖国贡献出自己一份力量。

驻纪第七纪检组 皇甫思阳
