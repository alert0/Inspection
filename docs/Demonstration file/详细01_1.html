该平台与“阳光扶贫”监管系统深度融合，使监管系统进一步扩容、下探、增效，擦亮打响“阳光扶贫”平台。采取“前台监督、后台执纪”的形式，全面收集主体信息、事项信息、专项信息，以身份证号为索引，设置关联、异常、矛盾三种比对模式，搭建精准扶贫和粮食直补数据分析比对模型，通过对海量的扶贫资金数据进行分析，找异常、找关系，重点围绕虚假扶贫、优亲厚友、截留私分、虚报冒领等情况，实现查询、比对、分析、预警、监督五大功能，让群众在“前台”举报有依据可寻、纪委在“后台”执纪有线索可查。平台运行以来，共梳理存量线索53条，立案查处31件，追缴违纪资金400余万元，平均初核时间节省了2/3，同时平台的建立对村干部也起到很好的震慑作用。
