江苏贾汪：“百千万”行动服务群众“零距离”

2017-10-30

阅读量：21

在党的教育实践活动中，贾汪区开展了“百千万”行动，在基层一线倾听意见、整改作风、破解难题、凝聚民心，切实打通联系服务群众的“最后一公里”。

百名“第一书记”驻村入户，知农忧谋农事

从处级干部和部门、乡镇“一把手”中选派了122人担任村党组织“第一书记”,力抓三件事。一是“强班子”。每月10次进村调研帮扶，帮助村班子建立健全“三会一课”、党员承诺评议、党员公益服务日等长效机制，带领所任职村努力争创“五好”党组织。二是“抓发展”。122名“第一书记”任职以来，以群众反映强烈的突出问题为突破口，新建村级幼儿园5个、修复灌溉桥涵闸15处,收运垃圾31600吨，清理积存垃圾及乱堆乱放23376处，栽植绿化树木9万棵，硬化道路169000米，改造和新建水冲厕所103座，修建下水道76130米,新建柴草堆放点120处。三是“保稳定”。积极帮助化解矛盾纠纷，走访慰问贫困户，参与村内决策重大事项，村内发生重大突发事件和矛盾纠纷时即时到场解决，保证任职村和谐稳定。目前，“第一书记”已走访群众6400户，召开村班子和群众座谈会380多场次，梳理村集体迫切解决的问题460余条，帮助镇村制定《发展规划》120份，协调落实资金580万元。

千名机关党员服务社区，解民忧宜民居

组织1000多名在城区居住的机关党员干部，以居住社区为活动阵地开展就地服务。一是组织有法律专长的党员干部开展义务法律咨询活动，为社区群众免费提供法律服务；二是深化文明礼仪、生活习惯、社会公德宣传，对不文明的行为进行制止和说服教育，协同社区做好各类矛盾纠纷的化解；三是组织具有医疗卫生专业知识的党员提供义务救护、上门服务、护理服务等，组织健康知识讲座，宣传医疗救护、卫生保健知识；四是以创建国家卫生城市为统领，加大小区绿化美化力度，完善基础设施配套，构建宜居社区，参与社区义务安全巡逻。目前已开展免费法律咨询63场，医疗宣传讲座85场，化解基层纠纷矛盾190件，推进实施环境集中清理800多件次。

万名党员干部结对帮扶，促民生暖民心

在全区范围内开展了万名党员干部结对帮扶万名贫困户活动。一是“亲情”结对。全区机关单位和各镇、街道、工业园区在编在岗党员，全区在编在岗教师党员，农村党员干部均参与活动。结对帮扶对象为农村建档立卡贫困户、重残户和孤儿。区领导每人结对帮扶4户，科级干部每人结对帮扶2户，其他机关工作人员及教师每人结对帮扶1户。二是真情帮扶。通过深入贫困家庭，了解实际情况，有针对性地分析原因，帮助贫困户克服思想压力，寻找致富出路，变“输血”为“造血”，通过学习技能、提升能力，实现致富增收。三是倾情“五送”。以送温暖、送政策、送技术、送项目、送文化为基础，针对不同帮扶对象具体困难，采取政策上帮、思想上引、资金上扶、技能上教等措施，全方位、多形式为扶贫对象做好生产和生活上的服务。目前全区万名党员干部通过多种形式落实帮扶资金220万元，帮助群众解决困难实事1750余件。
