贾汪区纪委召开全区纪检系统学习近平总书记精准扶贫四次重要讲话集中学习会

2017-07-04

阅读量：24

7月4日，贾汪区纪委按照王岐山书记在扶贫领域监督执纪问责工作电视电话会议上的要求，召开专题会议集中学习传达习近平总书记精准扶贫四次重要讲话精神。区委常委、纪委书记袁邦庆同志主持会议。

会议指出，王岐山书记在扶贫领域监督执纪问责工作电视电话会议上，对所有的纪检监察干部布置了一项任务，就是要认真学习习近平总书记在延安、贵州、宁夏、山西召开扶贫工作座谈会上的讲话。贾汪区全体纪检监察干部要以高度的政治自觉，认真学习领会讲话精神，在扶贫工作中找准纪检监察的定位，用监督执纪问责推动精准扶贫工作务实、过程扎实、结果真实。

会议要求，全区纪检监察组织必须把精准扶贫领域监督执纪问责作为今后工作的重中之重，采取“制度+科技+群众”模式监督好精准扶贫工作，要各尽其责，充分发挥自身职能，敢于反映和处理扶贫领域的腐败问题，加大问责督查力度，坚持失责必问、问责必严，为脱贫攻坚提供坚强的纪律保障。
