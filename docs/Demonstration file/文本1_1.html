贾汪区纪检监察巡察干部学习十九大报告精神交流会暨第一期“毛遂论坛”举行

2017-11-16

阅读量：31

11月16日，贾汪区纪检监察巡察干部学习十九大报告精神交流会暨第一期“毛遂论坛”举行，旨在进一步加强对十九大精神的学习、领会，进一步准确把握党的十九大确立的重大思想、重大判断、重大战略和重大任务。区领导袁邦庆出席会议并讲授专题党课。

　　第一期“毛遂论坛”上，10名纪检监察同志毛遂自荐，结合自身工作实际，分享了学习十九大报告精神的感悟和体会，为下一期“毛遂论坛”奠定了良好基础。

　　袁邦庆在讲授专题党课时结合实际，从把握党的十九大精神重心、过去几年发生的历史性变革、习近平新时代中国特色社会主义思想、新时代主要矛盾、新时代中国共产党历史使命、新时代基本方针和决胜全面建成小康社会等方面谈了学习体会。专题党课内容丰富，深入浅出，对十九大精神学什么、怎么学指明了方向，对区纪检监察巡察干部在今后的工作中，进一步准确理解和全面把握党的十九大精神具有指导性意义。

　　袁邦庆指出，全区纪检监察干部要坚持把自己摆进去、把职责摆进去，深入领会十九大报告精神实质，围绕“新时代”“新思想”“新矛盾”“新目标”进行全面把握，一段一段逐条学习，学深学透。学习中要联系实际、联系党情、国情深入思考党的十九大报告提出的一系列新思想、新论断。纪检监察干部一定要不忘初心，牢记使命，紧密结合纪检工作实际，坚持学以致用，不断提升自身能力素质，积极开拓进取，忠诚履行党章赋予的监督执纪问责职责，坚定自觉地把党的十九各项决策部署落到实处。
