挺纪在前   全面履职

扎实推进纪检监察信访举报工作


贾汪区纪委  监察局


2016年，贾汪区信访举报工作坚持把纪律和规矩挺在前面，把握运用“四种形态”，依纪依规、科学有效履职，主动发挥在惩治腐败、纪律监督、服务决策方面的作用，积极服务纪律审查和基层党风廉政建设。

一、深化三转，全力服务纪律审查

一是发挥主渠道作用。坚持信访案件及时呈批，快速转办。2016年，贾汪区纪委共受理群众信访举报195件次,属于检举控告类的175件次，占89.74%，本级自办123件，对下转交办72件。全年交办要结果、要情况信访件16件，转立案28件，信访室对所有交办信件全程跟踪处理，直至案结事了。

二是加大直查快办力度。积极发挥信访举报直查快办功能，做到快速出击、准确查处。2016年，信访室直查快办线索15件，立案8件，处理党员干部8人，实施信访监督38人次，平息群众来访7件，有力维护了群众利益和社会稳定。

三是突出信访件优质办理。不断加大督办审核力度，确保信访件按期优质办结。全年交办要结果信访件16件，及时督促承办科室按期报结，并向实名举报人反馈。

二、抓早抓小，促进信访监督提质增效

一是健全信访监督机制。按照四种形态要求，细化监督流程，让咬耳扯袖、红脸出汗成为常态，对反映的苗头性、倾向性以及一般性问题，充分运用信访监督进行函询说明、谈话提醒；发现轻微违纪问题，及时警示诫勉、纠错整改。进一步扩大了谈话函询覆盖面，对村级干部实施谈话函询。

二是提升信访监督质效。全年实施信访监督38人次，其中发放信访监督通知书17人次，信访谈话21人次。精准运用“四种形态”，起草扯袖关爱行动实施办法，查询函复组织部门、宣传部门、人大等多家单位提供党员干部在提拔使用、评先评优等党风廉政建设情况23件次,有效防止了带病提拔、带病评优。

三是加强信访信息分析。深度挖掘分析信访信息，加强预警研判。全年撰写报送信息25条，上级单位采用16条，其中，省纪委办公厅、省纪委信访室采用10条等。发表外宣6篇 ，被《中国纪检监察杂志》采用1篇，省、市纪委采用5篇等。全年编发信访简报4篇，信访专报4篇，其中专题报告《关于关于扶贫领域违纪问题的几点思考》，推动了全区扶贫领域专项督查的开展。年底，对十八大以来信访举报情况进行专题分析，形成《十八大以来全区纪检监察信访举报工作情况分析》报告，为2017年党风廉政建设及信访举报工作提供参考，推动解决群众反映强烈的突出问题。

三、五大机制，有效化解防范越级访

一是实行查否审核机制。对主要领导签批的、上级要结果的、巡视组交办的、向反映人反馈后不满意的、越级访的等五类信访件实行查否审核，经审查问题未查清的继续调查，切实提高办信质量。2016年，已经对7件信访件进行了查否审核，审核后重新启动调查程序的1起。

二是深化排查稳控机制。每月开展业务范围内信访问题全面排查，通过上下、内外、动静结合，对不稳定因素“清仓见底”，逐人逐案建立工作台账，着力摸清存量、厘清数量、提升质量；建立健全“五包一”稳控机制，重点人员均落实领导包案制度，做到重点人员有人管，稳控工作有人抓。

三是完善重复访化解机制。针对2014年以来重复信访举报情况，从信、访、网、电中认真筛选出11件重复信访举报件，作为集中化解任务,落实委领导及各监察室包案制度，建好重信重访户的档案，通过开展集中化解，截至今年1月底，集中交办的信访举报事项总体结案率93.9%，总体化解率75.7%。

四是构建信访监督预警机制。针对在实施信访监督过程中发现的典型性、倾向性问题，及时梳理总结，形成《信访专报》。对于问题线索中发现的违纪苗头或者轻微违纪行为的党员干部，及时扯袖子、咬耳朵，通过函询、约谈、诫勉谈话、信访调查等方式，快查快结，以督促整改一批、约谈警示一批、通报批评一批、立案查处一批。

五是严格责任追究机制。压实党委的主体责任和纪委的监督责任，凡发生业务范围内进京赴省越级访、集体访，实行信访绩效管理考核一票否决制度。每季度通报进京赴省越级上访情况，对于发生进京赴省越级上访的，约谈党委、纪委主要负责人，情况严重的，严肃追究主体责任和监督责任。2016年，分别对2家党委书记和纪委书记进行重点约谈，对越级访情况进行了全区通报。

四、2017年工作打算

一是建立信访举报双月专题会议制度。依纪依法受理、办理群众信访举报，引导群众正确行使权利，建立信访举报件专题会议制度，专题研究会办当期信访举报件的受理、办理情况，切实提高信访举报件办理质量。

二是提升信访举报信息化建设。按照“网上省纪委”平台建设要求，积极加强信访举报“大数据”开发汇集，做好分类录入涉及违反中央八项规定精神问题、“一把手”问题”、赴中央纪委监察部上访等信访举报数据的统计汇总，建立基本情况、热点领域、突出问题等相关数据统计制度。

三是提高信访举报分析水平。树立数字观念、分析观念、逻辑观念，加强信访举报综合分析，定期揭示党风廉政建设情况，服务领导决策，加强热点领域和重点问题分析，供领导掌握影响党风政风民风的突出问题，并向地区通报存在突出问题，及时采取相应对策措施。

四是全力化解防范越级访。积极预防和减少群众越级上访，做好十九大等重大节日期间维稳工作，定期开展矛盾纠纷专项化解行动，建立健全稳控机制，对于可能进京上访人员，特别是反复到上级纪委上访人员纳入一类管理，防止小问题演变成大问题。

五是开展惠民补贴违纪问题专项督办。加大对群众身边不正之风和腐败问题信访举报的交办督办力度，重点关注涉农惠农补贴、集体“三资”管理、征地拆迁等方面的信访举报，开展专项督办，认真梳理排查，及时组织交办，推动信访问题及时有效办理
