贾汪区纪检监察系统开展第四期“打铁还需自身硬”专项行动学习培训会

2017-06-23

阅读量：36

    6月23日，全区纪检监察系统开展“打铁还需自身硬”专项行动学习培训会，邀请专家现场授课。区领导袁邦庆出席会议。


　　今年3月，我区按照省、市纪委部署，组织开展了“打铁还需自身硬”专项行动，全面提升我区纪检监察干部理想信念、政治素质和业务水平，使队伍更加纯洁、肌体更有活力，工作更有方法和水平。


　　此次培训会邀请了市人民检察院副检察长、反贪局局长陈海鹰现场授课，从“重细节”、“会观察”、“讲方法”、“树形象”、“保安全”等五个方面，结合自身多年工作经验以及实际案例深入浅出为大家阐述了纪检监察工作的内涵和精髓。


　　袁邦庆指出，随着社会的发展、科学的进步，信息化、科技化已经成为纪检监察工作的必然趋势。此次培训内容丰富，既有理论又有很强的现实指导性和针对性。希望大家在下一步工作中结合此次讲课内容，主动转变工作理念和思路，把更多的先进理念融入到工作中，为贾汪大数据信息公开平台的启用做好思想准备和技术准备。
