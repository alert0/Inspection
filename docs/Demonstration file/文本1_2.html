参观淮海战役纪念馆心得体会

2017-03-24

阅读量：14

  3月24日，为更好地践行两学一做学习教育活动成果。贾汪区纪委机关党支部所属党员前往徐州参观了淮海战役纪念馆，并在纪念碑前重温了入党誓言、合影留念。在解说员的引导下，领略了淮海战役的全过程，令人震撼，感叹革命先烈非凡的胆识和英勇无畏的壮举。同时，对几百万踊跃支前的人民的行为，所付出的巨大代价感到赞叹，对陈毅同志的淮海战役是人民用小车推出来之说进一步有了认识。淮海战役我党我军以少于敌20万的兵力，（我方60万，敌方80万）打败国民党大军，关键是我方有几百万支前民工，担负了战争的运输、伤员救助、伤员转运，为连续作战提供了有力保障，使战役取得了胜利。人民是战争胜利的根本保证。敌方虽有80万大军，但缺少支援保障，战争又失得民心，信息来源不足，战力减弱。我军有人民的支援作战，能力成倍提高，其结果成为必然。人民是历史的创造者，只有人民才是取得胜利的力量源泉，毛泽东同志说过，一切胜利都来自人民，人民、只有人民才能战胜一切困难，是推动历史前进的动力。淮海战役再次证明了这一论断的正确。“得人心者得天下，失民心者失天下”，没有人民的支持帮助，任何事情都做不了，做了办不好。习总将人民的需求放在各项工作的首位，坚持“以人为本”的科学理念和发展观要求，把伟大“中国梦”贴近每个人，提高人民的的福址这一明智之举。淮海战役得宜于人民，对我党我军的认同和支持，并得到了认真执行。试想没有人民的支持将会走向何方，我们做任何工作都不要忘记人民的贡献，要时刻想着人民，时时处处为了人民，将人民的利益做为我党最大利益。把人民的需要做为党的工作目标，替人民担当责任。

常惟茜
