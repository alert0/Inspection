贾汪区全区巡察及派驻纪检干部培训班开班

2017-9-25

阅读量：15

　　9月25日，贾汪区全区巡察及派驻纪检干部培训班开班，全区110余位纪检监察干部学员参加培训。区领导袁邦庆出席开班仪式并为学员授课。


　　据了解，此次培训班为期三天，课程安排紧凑、内容丰富、形式新颖，将邀请徐州市纪委等有关领导，围绕信访举报、纪律审查、线索排查与初核、涉纪涉腐网络舆情处置、“四种形态”在纪律审查中的运用、“一准则两条例”解读等方面进行专题授课。


　　培训班开班当天，袁邦庆以“人心惟危，道心惟微；惟精惟一，允执厥中”十六字心经对“打铁还需自身硬”的启示为主题，为学员们讲授了第一课。他指出，反腐败斗争是一场持久战、攻坚战，纪检干部作为反腐一线的中坚力量和“排头兵”，必须把握当前形势，充分认识提高纪检监察队伍素质业务水平的重要性和必要性，不断提高纪检监察干部队伍的整体素质和工作水平，为推进党风廉政建设和反腐败斗争提供坚强的组织保障。
