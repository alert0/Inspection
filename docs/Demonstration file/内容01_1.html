2016年度，纪检监察五室在委、局领导的正确领导和关心支持下，坚持以党的十八大及十八届六中全会精神为指导，围绕中心，服务大局，扎实做好执纪监督，各项工作取得明显成效：


（一）加强业务学习，提高执纪办案水平。一是强化理论知识。深入学习贯彻新出台的《纪律处分条例》、《廉政准则》、《问责条例》、《关于新形势下党内政治生活的若干准则》、《党内监督条例》等党纪法规，不断强化自身的政治责任感和使命感。二是提高业务水平。主动开展纪检监察业务知识学习，积极利用“周末茶案”、高校学习等机会，增强科室执纪办案能力。三是培养锻炼新人。加强对科室纪检新兵的培养锻炼，促其尽快熟悉业务，提高其独立办案能力。我室负责联系的汴塘镇案件目标数19件，完成21件，超额完成11%；潘安湖办事处案件目标数12件，完成17件，超额完成42%。


（二）严格执纪，强化纪律审查。2016年案件目标33件，实际完成35件，其中科级干部案件目标2件，完成2件；经济案件目标16件，完成19件；查处违反“八项规定”案件2件，问责2人，函询1人，有效维护风清气正的政风环境。一是强化查信办案。2016年度共收信访件14件，信访转立案9件；省委巡视组交办件11件，其中转立案2件，党员除名1人，免职1人。有效化解辖区信访矛盾。二是深挖案源线索。在充分发挥信访主渠道作用的同时，积极从个案中深挖其他案件线索，深挖同类问题，及时与审计、财会、司法机关沟通，认真开展失地保障专项问题清理，共挖掘各类线索22条，为案件办理提供良好保障。三是严查基层腐败。2016年，重点查办塔山镇聂庄村、张场村、葛湖村以及潘安湖办事处潘安村、唐庄村等发生在群众身边的违纪问题15件，有效维护党的形象。四是践行“四种形态”。 把全面从严治党落实到对党员的教育监督管理全过程，真正做到执纪在前，持续保持惩腐高压态势。2016年度全年共约谈各级干部12人、函询1人。


（三）认真谋划，强力推进廉政建设。加强对十八大以来区委区政府重大决策部署执行情况的监督检查，特别是对区重点工作、重要民生工程等重大决策部署执行情况的监督检查，狠抓落实、务求实效，认真参与区优化营商环境，“三清示范区”建设工作，严格监督农村基层党风廉政建设，督促村务公开，强化“三资”监管及“勤廉双述”落实工作，促进全区工作各项政策和任务科学、廉洁、高效落实，确保权力行使安全、资金运用安全、项目建设安全和干部成长安全。
